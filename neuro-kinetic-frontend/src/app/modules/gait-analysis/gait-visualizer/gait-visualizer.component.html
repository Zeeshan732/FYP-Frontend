Gait Analysis

Screening workspace for clinically observed gait parameters.

About this assessment

Gait analysis helps identify movement abnormalities commonly associated with Parkinson's disease. Enter clinically observed gait values manually, or import a standard-format CSV — then review, adjust if needed, and run analysis. Results support clinical discussion only and are not a substitute for formal diagnosis.
