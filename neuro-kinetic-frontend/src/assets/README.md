# Assets Structure

Use this structure for media files in NeuroSync.

## Folders

- `logos/brand`  
  Primary brand marks (full logo, mark-only, light/dark variants)
- `logos/partners`  
  Partner or organization logos
- `icons/ui`  
  Product UI icons (`.svg` preferred)
- `icons/social`  
  Social platform icons
- `images/backgrounds`  
  Hero and section backgrounds
- `images/photos`  
  Real photography content
- `images/placeholders`  
  Temporary or fallback images
- `illustrations/auth`  
  Auth/login/signup illustrations
- `illustrations/empty-states`  
  Empty state graphics
- `avatars/defaults`  
  Default profile avatars
- `documents`  
  Static downloadable files (PDF guides, etc.)
- `videos`  
  Marketing or demo videos
- `lottie`  
  Lottie animation files (`.json`)

## Naming Convention

- Use lowercase and kebab-case.
- Include context and variant when needed.
- Examples:
  - `logo-neurosync-light.svg`
  - `hero-neurology-background.webp`
  - `icon-finger-tap.svg`

## File Format Guidance

- Logos/icons: `.svg`
- Photos/backgrounds: `.webp` (preferred), `.png` fallback
- Documents: `.pdf`
- Videos: `.mp4`
